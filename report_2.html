การเขียนสะท้อนตนเองแบบการเรียนรู้ผ่านประสบการณ์ Week # 1
(Recursive) ประสบการณ์ในสัปดาห์นี้มีการเชื่อมต่อกับประสบการณ์หรือแผนการที่ผ่านมาหรือไม่? (ได้เรียนรู้การใช้ css เบื้องต้น)
(CE) อะไรคือประสบการณ์ที่ผ่านมาของคุณ ? (เรียน html ในสัปดาห์ที่ 1 )
(RO) อะไรคือประสบการณ์ที่สำเร็จและล้มเหลว ? (เข้าใจการทำงานของการใช้ css เพื่อออกแบบเว็ปไซต์ และดูองค์ประกอบ)
(AC) คุณได้เรียนรู้อะไรจากประสบการณ์ ? (การเลือกใช้สีให้เหมาะกับเว็ปไซต์ เพื่อความสวยงาม)
(AE) คุณจะได้นำสิ่งที่เรียนรู้จากประสบการณ์ ไปใช้อย่างไรต่อไป ? (สามารถนำความรู้ที่ได้รับไปออกแบบเว็ปไซต์ของตัวเอง)
